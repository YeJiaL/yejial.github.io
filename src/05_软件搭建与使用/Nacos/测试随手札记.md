## 关闭

```bash
nginx -s stop 立即停止nginx,不保存相关信息
nginx -s quit 正常退出nginx,并保存相关信息
nginx -s reload 重启(改变配置后,需重启)
taskkill /f /t /im nginx.exe 彻底关闭
```

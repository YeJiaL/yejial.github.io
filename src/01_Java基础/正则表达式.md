

# 正则表达式实现与或非



我们先虚拟两个表达式PatternA和PatternB，分别校验条件A和条件B

一 匹配字符串同时满足条件A和条件B

(?=PatternA)PatternB

二 匹配字符串满足条件A,但不可满足条件B

(?!PatternB)PatternA

三 匹配字符串或者满足条件A或者满足条件B

(PatternA|PatternB)
